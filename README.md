# 项目简介

## 核心功能
- 登录注册
- 选课退课
- 查成绩
- 生成成绩单
- 查排名
- 成绩管理
- 学生管理
- 课程管理

## 适用场景
- 教师可使用系统进行成绩管理、课程管理等操作。
- 学生可使用系统进行选课退课、查成绩、生成成绩单等操作。

## 技术特点
- 采用 Python 编程语言进行开发。
- 使用 SQL 数据库进行数据存储。
- 提供演示视频，方便用户了解和使用系统。
